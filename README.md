# Borracharia 24h Só Chamar

Site profissional gerado por **Lead Machine** — Next.js 14 + Tailwind CSS.

## Nicho
Borracharia — Osasco

## Stack
- Next.js 14 (App Router)
- Tailwind CSS 3
- React 18
- GSAP + ScrollTrigger para motion cinematica
- Google Fonts (Inter)

## Rodar local
```bash
npm install
npm run dev
```

## Deploy
```bash
npm run build
```
